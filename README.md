# szamowski.dev

> Software that feels native, wherever it runs.

The personal portfolio of [Maciej Szamowski](https://szamowski.dev), a Fullstack Developer based in Warsaw. It is a focused home for work across Swift, React, Next.js and TypeScript.

![Deep navy canvas with a cyan six-point asterisk](public/brand/szamowski-hero.png)

## Built for the work

The site puts the projects first: Hora, Prismatic, Copa City and GNOME Tray Toggle. The visual system stays intentionally quiet, with a deep navy canvas, a single cyan mark and small moments of movement where they add character.

- **Next.js 16** with the App Router
- **React 19** and **TypeScript**
- Responsive layout with accessible navigation and controls
- A progressive hero shader that respects reduced-motion and reduced-data preferences
- Consent-first analytics and marketing integrations
- Generated metadata, Open Graph images, sitemap, robots and web manifest

## Run it locally

```bash
npm install
npm run dev
```

Open [localhost:3000](http://localhost:3000).

## Quality checks

```bash
npm run lint
npm run typecheck
npm run build
```

## Project map

```text
app/           Routes, metadata and the page composition
components/    Reusable interface, consent and visual components
lib/           Site configuration and client-side utilities
public/        Project media and brand assets
```
